# HR-Attrition-Analytics-Dashboard-using-PowerBI
Developed an HR Attrition Analytics Dashboard using Power BI, Power Query, and DAX to analyze employee turnover, workforce trends, and key HR metrics through interactive visualizations and KPI reporting.

# IBM HR Attrition Analytics Dashboard

## Tools Used

1. Power BI
2. Power Query
3. DAX
4. Excel
5. Data Cleaning
6. Data Transformation
7. Data Visualization

## Project Overview

1. Built an HR Attrition Analytics Dashboard using the IBM HR dataset to analyze employee turnover, workforce demographics, and attrition trends.
2. Created and tracked key HR metrics including Employee Count (1,470), Attrition Count (237), Attrition Rate (16.1%), Average Age (37 years), Average Salary (6.5K), and Average Tenure (7 years).
3. Used Power Query to clean and transform employee data and DAX to create calculated measures and KPIs for reporting.
4. Developed interactive dashboards to analyze attrition across departments, job roles, age groups, education fields, salary slabs, and years at the company.
5. Explored workforce trends to identify employee segments with higher attrition and understand factors contributing to employee turnover.

## Key Insights

1. The overall attrition rate was 16.1%, with 237 employees leaving out of a total workforce of 1,470 employees.
2. Attrition varied across departments, job roles, and age groups, helping identify areas with higher employee turnover.
3. Salary levels and employee tenure showed noticeable differences in attrition patterns across the workforce.
4. Workforce demographics and employee experience data provided useful insights into employee retention trends.
5. Interactive reports enabled detailed analysis of attrition drivers and workforce metrics.
